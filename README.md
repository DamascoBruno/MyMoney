# My Money 💰

Um aplicativo de organização financeira pessoal desenvolvido em Python, focado em registrar entradas e saídas de forma simples e eficiente, utilizando armazenamento local em JSON.

---

## 🚀 Funcionalidades (MVP)

*   **Registro de Receitas:** Adicione suas fontes de renda de maneira rápida.
*   **Registro de Despesas:** Controle seus gastos categorizando suas saídas.
*   **Armazenamento em JSON:** Dados salvos localmente em um formato leve e de fácil leitura, sem a necessidade de configurar um banco de dados complexo.
*   **Visão Geral Financeira:** Cálculo automático do saldo atualizado com base nas entradas e saídas registradas.

---

## 🛠️ Tecnologias Utilizadas

*   **Linguagem:** Python
*   **Armazenamento de Dados:** JSON (Manipulação via biblioteca padrão `json` do Python)

---

## ⚙️ Pré-requisitos e Instalação

Para rodar este projeto na sua máquina, você precisará ter o [Python](https://www.python.org/downloads/) instalado.

1.  **Clone o repositório:**
```bash
    git clone [https://github.com/SEU_USUARIO/my-money.git](https://github.com/SEU_USUARIO/my-money.git)
    ```
2.  **Acesse o diretório do projeto:**
```bash
    cd my-money
    ```
3.  **Execute a aplicação:**
```bash
    python main.py
    ```
    *(Nota: Substitua `main.py` pelo nome real do arquivo principal do seu projeto, caso seja diferente).*

---

## 💻 Como Usar

Ao iniciar a aplicação, você terá acesso a um menu interativo no terminal (ou interface, caso tenha implementado) onde poderá escolher entre:
1. Adicionar uma nova receita.
2. Adicionar uma nova despesa.
3. Visualizar o extrato e o saldo atual.
4. Sair do aplicativo.

Os dados serão salvos automaticamente no arquivo `.json` gerado no diretório do projeto.
